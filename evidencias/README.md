## Comandos Executados
ls - listagem de ficheiros e diretorios
pwd - mostra em que diretorio o utilizador se encontra
cd - utilizado para mudar de um diretorio para o outro
apt install - comando usado para instalar aplicativos/software/servicos que queremos usar. 
nano - usado para editar ficheiros no linux.
neofetch - utilizado para ver informacoes sobre o sistema operativo em uso. 
cat - comando serve para ver conteudo dentro de um ficheiro. 

## Diferenca entre VM, VPS e infraestrutura em nuvem
VM - Maquina Virtual e uma forma de virtualizar um computador com intuito
de dividir os recursos da maquina de modo executar um outro servico o sistema operativo. 

VPS - Virtual Private Server e um servidor disponibilizado por uma empresa de
hospedagem de modo que o cliente tem controle total do servidor nao divindo 
recursos com nenhum outro cliente/maquina.

Infraestrutura em nuvem conjunto de componentes de hardware e 
software necessários para hospedar e
executar aplicativos, armazenar dados e gerenciar redes na internet.
