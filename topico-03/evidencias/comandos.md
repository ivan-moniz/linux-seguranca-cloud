# Comandos Utilizados 

## Apresenta-se abaixo todos os comandos utilizados na realizacao da dinâmica

Atualizacao de pacotes
`sudo apt update`

atualizacao do sistema
`sudo apt upgrade`  

Instalacao do apache
`sudo apt install apache2`

Habilita o firewall para protecao da nossa maquina
`sudo ufw enable`

Permissao do firewall para o acesso remoto
`sudo ufw allow ssh`

Permissao do acesso via http - porta 80
`sudo ufw allow 80/tcp`
